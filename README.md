# NTBSS-Force-Save
a tool for Naruto to Boruto: Shinobi Striker to allow the game to dump your online save to disk and load a modified save from disk


1-First replace your old Naruto.exe inside(D:\steam\steamapps\common\Naruto To Boruto) with the modified one (backup ur old one).

2-Then go to (D:\steam\steamapps\common\Naruto To Boruto\NARUTO\Binaries\Win64) and paste everything that is inside the folder (ues4ss files) to there.

3-lunch the game and load into a lobby first, you must be fully inside the game( not the title screen)

4-Run the gui exe or the python script

4-then choose dump or upload save. If you uploaded a save, once it complete go back to the tile screen then back to game for it to update.
